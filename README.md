### Prerequisites

- **Bun** >= 1.0.0 ([Install](https://bun.sh))
- **Google Gemini API Key** ([Get one free](https://makersuite.google.com/app/apikey))

### Setup in 2 Minutes

1. **Clone and Install**:
   ```bash
   git clone <your-repo-url>
   cd codalyn
   bun install
   ```

2. **Start Development**:
   ```bash
   bun run dev
   ```

3. **Open Browser**:
   - Visit [http://localhost:3000](http://localhost:3000)
   - Enter your Gemini API key
